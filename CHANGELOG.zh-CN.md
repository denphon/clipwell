# 更新日志

本文件记录项目的所有重要变更。

本更新日志按发布 tag 组织，基于 git 提交历史整理。

> English version: [CHANGELOG.md](CHANGELOG.md)

## 2.0.0.5 - 2026-06-25

### 新增
- 历史列表中的纯文本条目新增**智能内容识别**：纯颜色值（`#rgb`/`#rrggbb`/`#rrggbbaa`、`rgb()`/`rgba()`、`hsl()`/`hsla()`）显示色块预览，`http`/`https` 链接高亮且可点击在默认浏览器打开，疑似代码的文本以等宽代码块呈现。
- 设置中心新增独立的**剪贴板**分组，**富文本默认粘贴格式**与**富文本/HTML 预览方式**由「常规」移入该分组。

### 变更
- 便携版安装包改名为 `Clipwell-<version>-portable.zip`，以便与安装程序区分。

## 2.0.0.4 - 2026-06-23

### 新增
- **富文本（HTML/RTF）支持**：剪贴板条目现可捕获并还原 HTML/RTF 格式，在列表中展示格式化预览，并在右键菜单提供**以纯文本粘贴**。
- 设置中心新增**富文本默认粘贴格式**（富文本/纯文本）与**富文本/HTML 预览方式**（富文本/纯文本）两项设置。
- 打包时在便携版 zip 之外额外生成 **NSIS 安装程序**（`Setup.exe`）。

### 变更
- 图片改用无损的 **PNG** 剪贴板格式：捕获与还原不再重新编码（保留透明度），**保存为文件**导出由 `.jpg` 改为 `.png`。
- 点击或复制条目不再刷新其时间戳。

### 修复
- 修复窗口隐藏后重新打开时右键菜单残留的问题。

## 2.0.0.3 - 2026-06-22

### 新增
- 设置中心「通用」新增**窗口位置**选项，可选择历史窗口打开时出现的位置：跟随鼠标、屏幕中间或屏幕右下方。
- 设置中心「更新」新增**网络代理**配置，更新检查与下载可经由指定代理。

### 变更
- 历史窗口点击任意非输入组件后，焦点会自动返回搜索框。
- 移除 exe「解锁」模式及相关逻辑。

## 2.0.0.2 - 2026-06-18

### 修复
- `lastUpdateCheck` 不再保存在 `config.json` 中。该运行时状态改存于独立的 `update_state.json`，首次启动时会自动迁移 `config.json` 中遗留的旧值。

## 2.0.0.1 - 2026-06-18

### 变更
- 产品更名：由「ClipHistory」改为 **Clipwell**。可执行文件、窗口标题、托盘提示、关于对话框及全部内置翻译均启用新名称。首次启动时自动迁移原有设置、剪贴板历史、收藏项和缓存图片，并沿用原开机自启动项，无需任何手动操作即可继续开机启动。

## 1.1.0.8 - 2026-06-18

### 新增
- 新增内置软件自更新：自动检查公开 GitHub 仓库的新版本，下载最新发布包并原地替换正在运行的程序，随后提示重启。设置中新增「更新」分区，显示当前版本、可选择检查频率（手动 / 启动时 / 每天 / 每周），并提供一键「立即检查」。
- 托盘菜单新增「检查更新」入口，点击后打开「更新」分区并立即开始检查。

## 1.1.0.7 - 2026-06-18

### 新增
- 为公开 GitHub 仓库新增 MIT 许可证与英文 `README.md`（原中文版迁移至 `README.zh-CN.md`）。
- 新增 GitHub Actions 工作流：推送版本 tag 时自动构建、打包并发布带 zip 产物的 GitHub Release，另含针对 push 与 PR 的 CI 构建校验。

## 1.1.0.6 - 2026-06-18

### 变更
- 历史、设置、关于三个无边框窗口改用基于鼠标捕获的手动拖拽，替代系统模态移动循环；跨不同 DPI 显示器拖拽时内容可实时刷新。

### 修复
- 修复右键菜单「添加备注」无效的问题，并将备注持久化到磁盘，窗口重新加载后仍保留。
- 修复设置窗口与关于窗口跨不同 DPI 显示器后物理尺寸变化的问题，现与历史窗口一致保持物理尺寸不变。

## 1.1.0.5 - 2026-06-17

### 新增
- 新增托盘菜单的「重启程序」命令。
- 新增剪贴板文本与图片的「保存为文件」功能：图片导出为 `.jpg`，文本按内容自动识别扩展名（`.txt`、`.md`、`.py`、`.js`、`.c`），并在设置中提供可配置的导出保存路径。

### 变更
- 通过 `useMemo` 与 `useDeferredValue` 优化历史搜索性能。
- 显示历史窗口时，列表自动滚动回顶部，并将当前标签重置为「所有」。

### 修复
- 新增 `vite-env.d.ts`，修复 React UI 中 CSS 导入仅在编辑器中报的 TypeScript 诊断。
- 增强重启逻辑，使其在 `unlock` 模式下更安全，并保留启动参数。
